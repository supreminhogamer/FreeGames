# 🚀 MoreGames - Windows Fix (Geode v5.6.1)

![GitHub Release](https://img.shields.io/github/v/release/waolth/FreeGamesFixed?color=green&label=Download%20Fixed%20Mod)
![GitHub Stars](https://img.shields.io/github/stars/waolth/FreeGamesFixed?style=social)

This is a fixed and updated fork of the **MoreGames** mod, specifically compiled to resolve compatibility issues on Windows with the latest Geode versions.

---

## 🛠️ What's Fixed in this Release?

Many users encountered the following error when trying to use the mod:
> `Unknown Error: Unable to find platform binary under the name "bitz.moregames.dll"`

This happened because the previous releases lacked the compiled binary file for Windows. Additionally, the mod was built using an outdated SDK version.

**Improvements in this Fork:**
* **Native Windows Compilation:** I have manually compiled the missing `.dll` binary, allowing the mod to load correctly on Windows systems.
* **SDK Update:** The source code has been updated and tested for **Geode SDK v5.6.1** (previously v5.5.3).
* **Asset Integrity:** Ensured all textures and level files are correctly packaged within the `.geode` file.

## 📖 Installation Guide

1.  Download the `bitz.moregames.geode` file from the link above.
2.  Navigate to your Geometry Dash installation folder (usually managed by Steam):
    `C:\Program Files (x86)\Steam\steamapps\common\Geometry Dash\geode\mods`
3.  Drop the downloaded `.geode` file into the `mods` folder.
4.  Restart Geometry Dash and enjoy!

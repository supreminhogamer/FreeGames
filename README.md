### 🛠️ What's Fixed in this Release

This is a custom compiled build to fix the recent issues with the Windows version of the mod.

* **Missing Binary Fix:** Compiled the missing `bitz.moregames.dll` for Windows natively. This completely resolves the dreaded `Unknown Error: Unable to find platform binary under the name "bitz.moregames.dll"` popup when launching the game.
* **Geode SDK Update:** Updated the `mod.json` file to support the latest **Geode v5.6.1** SDK (previously stuck on v5.5.3), preventing compatibility errors during load.

### 📥 How to Install
1. Download the `bitz.moregames.geode` file from the **Releases**.
2. Go to your Geometry Dash installation folder (usually `C:\Program Files (x86)\Steam\steamapps\common\Geometry Dash\geode\mods`).
3. Drop the downloaded `.geode` file into that folder (replace any older version if asked).
4. Restart Geometry Dash and enjoy!
